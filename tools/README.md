# ffmpeg executable for MacOS or Linux

## Ubuntu 20.04 LTS

1. `sudo apt install ffmpeg`
2. `cp $(which ffmpeg)` ./src/HelperApp/Tools`

## Debian 11

1. `sudo apt install ffmpeg`
2. `cp $(which ffmpeg)` ./src/HelperApp/Tools`

## MacOS Ventura

1. `brew install ffmpeg`
2. `cp $(which ffmpeg)` ./src/HelperApp/Tools`
